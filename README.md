# Finance-api
An REST API which help to manage finance and create statistic.

## Documentation
Swagger (development environment): http://localhost:8080/swagger-ui/#/

## Technologies
- Java 17 (Java Development Kit - JDK: 17.0.6)
- Spring Boot 2.5.1
- Maven
- JUnit 5
- PostgreSQL 14
- Swagger 3.0.0
